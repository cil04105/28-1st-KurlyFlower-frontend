## 적용 기술 및 구현 기능

### 적용 기술

> - Front-End : React.js, react-routder-dom
> - Back-End : Python, Django web framework, Bcrypt, My SQL
> - Common : Git-Hub, slack, trello

### 구현 기능

#### 로그인페이지
- 로그인/회원가입

#### 메인페이지
- 상품 광고 슬라이드
- 네비게이션 메뉴
- 카드 및 기타 상품 소개

#### 상품 리스트 페이지
- 컴포넌트 재활용
- 쿼리스트링을 사용한 페이지 변경
- 상품 정렬 기능
